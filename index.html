ACCESS + AI INTEGRATION LAYER
FASTAPI
Backend API Layer
  • Exposes trusted analytics endpoints
  • Reads from PostgreSQL views/tables
  • Supports dashboard, BI tools, and AI agents
  • Returns JSON responses
TEXT-TO-SQL
Text-to-SQL Agent
  • Converts user questions into SQL
  • Uses approved schema context only
  • Runs SELECT-only queries
  • Blocks unsafe SQL and restricted tables
AI AGENTS
AI Agent Integration
  • Routes user questions
  • Calls approved tools/views
  • Summarizes results with evidence
  • Prevents raw-data hallucination
DASHBOARD AI
Streamlit + Ask AI
  • Interactive dashboard
  • Ask AI inside dashboard
  • Uses backend/agent outputs
  • Explains current dashboard data
